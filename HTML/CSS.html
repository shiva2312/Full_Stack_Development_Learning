In this we will learn about inclusion of CSS in HTML structure.

Basically there are three method by which we can include css in our structure:-

  • Internal CSS:-

    Here basically we include css property and value in header section of html. For example:- <style< body {background : green} <style<

  • External CSS

    • This means that we have defined the css property in some other file and from there we render that file in the index page or the page where we want to use that file. For example :- <link rel = stylesheet; href = styles.css<
