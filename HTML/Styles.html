In this section we'll learn how to use style attribute or Inline css

Note here we would be noticing that the background colur of the body is found to be greenish this is because I have used Inline CSS of background property with green value.

To use Inline CSS or style attribute for a particular tag the syntax is:-
<tagname style = "property : value"> <tagname>
where property is CSS property(color) and value is CSS value(red)

Here I have used style attribute with color Red. Simple what we can do for using style attribute we will use style="color:Red"
