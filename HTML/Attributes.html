In this we will understand about the HTML attributes

  • All HTML elements can have attributes
  • Attributes provide additional information about elements
  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs like: name="value"
  • Now let see the example of attributes:-
    1. img tag may have href="Source of file" where href can be considered as attributed
    2. suppose if we want to define the width and height of the image in the img tag then we can use width and Height attributes.

With this we have understood the Attributes functionality in HTML. Now we will see how we did the points in bullets points and number list:-

  1. To maintain a numbered format for the points, it is necessary to utilize the ol tag, which stands for "ordered list," and nest the individual list items within li tags.
  • To maintain a bullet point format for the items, the ul tag, which represents an "unordered list," should be used. Each individual list item should be enclosed within li tags to create the desired bullet list structure.
